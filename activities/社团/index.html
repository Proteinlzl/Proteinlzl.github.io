# 魅力的社团活动

吉林一中社团联合会是以 “多元智能理论” 为指导，由校团委社团部进行全面管理，旨在丰富我校学生课余生活，发展学生兴趣特长，培养学生智力广泛性与能力多样性的学生团体。

学校团委每学年初举行社团纳新大会，根据学生兴趣爱好自主组建，依据《吉林一中社团管理制度》、《吉林一中社团联合会章程》审批管理，现有社团涉及自然科学、社会科学、文艺娱乐、体育发展四大领域多个方面。

学校高度重视学生社团活动的发展，在学校领导的统筹规划下，学生社团以社团选修课的形式亮向新学年，为每个社团提供固定活动场所，积极为学生社团风采展示提供平台。学生社团主要通过组织特色活动、社团展示周、校园文化艺术节、校外实践基地、对外交流活动五个主要载体进行展示。

多年来，几十个社团炫异争奇，百花盛放，成为了学生课余生活的一道亮丽风景线。
