# 分类简介

百余年中，一中学子多人成为国家的栋梁之才。

  • 有马骏、王希天等革命家；
  • 有王以哲、姚新一、周建华、李光汉、张海慧等革命烈士；
  • 有王家祯、庄尚严、郭峰、于克、祝铭山、石忠信等政界要员；
  • 有王国香、周廉、王泽山、岳希新、马宗晋、张莘夫、苏士权、范士合等科技精英；
  • 还有穆木天、王永芳、关广志、韩笑、王宗汉等著名诗人、体育健将、画家和作家。

可谓英才辈出、桃李天下。
