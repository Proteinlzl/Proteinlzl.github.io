02. 孙一龙

# 简介

孙一龙是西昌卫星发射中心技术部一名系统工程师，也是吉林一中 2007 届 10 班的毕业生

# 家庭

# 家人

# 爷爷

孙协华是孙家第一代的保家卫国人。
1931 年出生于湖南农家，15 岁跟着共产党闹革命，19 岁考入长沙高级工程兵学校，入伍成为一名军人。
21 岁参加抗美援朝。
回国后，随部队进驻吉林把家安在了东北后，又跟随部队调转多地。
直至 70 年代末，部队调整转业。

# 父亲

孙江是孙家第二代的从医报国人，受孙协华的影响考入了第四军医大学，成为了一名军医。
不仅遂了守护父母健康的心愿，还圆了自己从小的军旅梦想，还为孙家找到了同为军医的媳妇贾丽茹。
90 年代初为照顾双方年迈的父母，两人陆续转业回到吉林在地方医院继续从医报国事业。

# 影响

孙一龙很小的时候就喜欢听爷爷讲抗美援朝的故事，戴爸爸妈妈的大檐帽
在爷爷和爸爸妈妈的影响下，怀着对国防科技事业的向往之情，高考时以优异成绩考入双一流大学的计算机系。
上大学不久他意外地发现自己所在的校区是原来的工程兵学校，正是爷爷参加抗美援朝前学习生活的地方。
这份机缘巧合让他更加坚定了知识报国的信念。
大学毕业后他来到了川西大凉山里的西昌卫星发射中心，成为了一名航天人。

# 离校前经历

爷爷在电话里问他为什么要选离家那么远的单位，他说 “您总说祖国要强大，孙家的子孙要站出来，我就来了。”
电话那头爷爷乐得笑出了声，说为他感到骄傲而这句话成了埋进孙一龙心底的一颗糖，在航天追梦迷茫或困顿时给予他温暖和鼓励

# 2012 年

他开始参与文昌发射场加注供气软件研发，研发关键时期爷爷病重住院为了不影响他工作父母没有告诉他，爷孙俩竟错失了最后一面

# 2014 年

带着 “大火箭” 的梦，孙一龙从西昌调往文昌，负责文昌发射场低温动力加注系统软件和设备调试工作。
同年底，女友赵兴娜为爱辞去北京研究所的工作，来到海南加入文昌航天的创业方阵，选择和他一起追梦航天。

# 2017 年

“天舟一号” 任务发射前夕，孙一龙因排除重大故障荣立个人二等功火箭。
点火前发射场区突然雷暴连连，赵兴娜和气象团队一起准确预报判定晴朗窗口，确保了火箭准时点火。

# 2020 年

是抗美援朝出国作战 70 周年，也是西昌卫星发射中心组建 50 周年。

# 评价

孙一龙一家三代的红色传承与战斗延续正是这段历史的生动映照，更是伟大抗美援朝精神 “两弹一星” 精神西昌航天精神的浪漫传承
西昌航天 50 年一代代西昌航天人在爱国情・强国梦的感召下不畏风浪、勇毅登攀、接续奋斗，在广袤太空书写了中国奇迹。

# 精神不老，梦想长青

孙一龙和团队一起正为中心下半年高密度任务征战

“我将带着爱国情和强国梦继续冲锋”

孙一龙的话语中充满自信。
